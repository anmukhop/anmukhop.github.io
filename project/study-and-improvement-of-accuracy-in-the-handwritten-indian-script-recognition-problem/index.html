Study and improvement of accuracy in the handwritten Indian script recognition problem

Anirban Mukhopadhyay, Ram Sarkar
Dec 5, 2019
Widely used Indian scripts

Used texture and frequency-based features along with CNN. Performed classifier combination to achieve high accuracy in the script identification task.
